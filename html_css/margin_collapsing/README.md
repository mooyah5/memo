# 마진 겹침 (마진 상쇄, Margin Collapsing)

2개 이상 블록 요소의 마진이 겹칠 때, 한쪽 값만 적용하는 브라우저 나름 렌더링 규칙

-   더 큰 마진 값으로 상쇄함.
    -   인접한 형제 블록 간 위/아래 마진이 겹칠 때
    -   부모와 첫/마지막 자식 간 마진이 겹칠 때 (단, 부모에 `padding`, `border` 등이 있으면 경계가 있다고 보고 겹치지 않음.)
    -   빈 컨테이너(`height=0`)의 위/아래 마진이 겹칠 때

### 해결 방법

마진 상쇄는 인접 요소가 `block-level` 요소일 경우에만 적용된다.

-   `display: inline-block;`
    -   부모에만 인라인블록 적용 시, `overflow: hidden` 도 같이 필요
-   `display: flex || grid;`
